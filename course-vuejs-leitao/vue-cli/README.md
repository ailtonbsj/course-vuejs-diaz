# Commands used

```bash
# Cria novo projeto
vue create projeto-simples

# Entra na pasta do projeto
cd projeto-simples

# Roda servidor
npm run serve

# Construe app na pasta dist
npm run build
```

```bash
# Cria novo projeto
vue create projeto-plugins

# Entra na pasta do projeto
cd projeto-plugins

# Roda servidor
npm run serve

# Electron plugin
vue add electron-builder

# Roda app como desktop
npm run electron:serve

# Electron plugin
vue add vuetify
```
